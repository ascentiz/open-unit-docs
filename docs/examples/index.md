# Examples

Examples show how developers might structure real applications on Open Unit without assuming a finalized SDK or package layout.

## Example Design Goals

Examples in this repository should:

- demonstrate a narrow, realistic task
- make the safety boundary obvious
- be easy to port into lab or partner codebases
- include enough context to be educational without being oversized
