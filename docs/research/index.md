# Research

Ascentiz Open Unit is intended to be useful not just for product development, but also for structured robotics and rehabilitation research.

## Research-Oriented Priorities

The platform aims to support:

- reproducible experiment setup
- access to high-quality telemetry
- clear boundaries for safe custom control logic
- collaboration between internal teams and external labs
